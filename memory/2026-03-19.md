### Lessons from this shift
- `Date.now()` in async server component = impure function lint error, even in helper functions. Always use `new Date().getTime()`.

### Lessons
- **Docker build ≠ local worktree build.** The Dockerfile runs `npm run build` at monorepo root. A local `npm run build` in a subdirectory can succeed even if there are cross-package type errors. When modifying types or interfaces, check the full build.
- **Type width mismatch:** `null` and `undefined` are not interchangeable in TypeScript strict mode. If an interface says `error?: string`, the mapped value must be `string | undefined`, not `string | null`. Safe pattern: `error: agent.error ?? undefined`

### Self-improvement
- Add `?? undefined` when mapping API responses to interfaces with optional (not nullable) fields
- When finishing a worktree, check if any modified file imports from a shared type/interface file and do a cross-reference check
